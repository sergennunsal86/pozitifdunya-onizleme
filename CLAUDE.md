# Pozitif Dünya — Web Tasarım Projesi

## Proje Bilgileri
- **Marka:** Pozitif Dünya / Gülhan Toktay
- **Sektör:** Eğitim, NLP, Koçluk, Kişisel Gelişim
- **Hedef Kitle:** Ağırlıklı kadın, 25-55 yaş, içsel dönüşüm arayan bireyler
- **Teknoloji:** WordPress + Hello Tema + Elementor + JegElementor Kit
